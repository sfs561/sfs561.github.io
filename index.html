What is a CPU cache?

A CPU cache is a small, very fast type of memory inside the processor. It stores copies of data the CPU uses often, so it doesn't have to wait for slower RAM. This makes programs run faster.

Intro to normalisation in databases

Normalisation is the process of organising tables in a database so that each table focuses on one topic and there is minimal duplicated data. This keeps data consistent and easier to maintain.
